# Frontend Learning Projects

这个仓库记录了从原生网页到 Vue、再到 Spring Boot 前后端联调的学习过程。

## Project Order

1. `01-vanilla-personal-site`：使用原生 HTML、CSS 和 JavaScript 搭建的个人网站。
2. `02-vue-cdn-demo`：不使用构建工具的 Vue 3 入门练习。
3. `03-vue-vite-demo`：使用 Vue 3 和 Vite 的工程化练习。
4. `04-vue-task-planner-frontend`：学习计划管理器的 Vue 前端。
5. `05-vue-task-planner-backend`：学习计划管理器的 Spring Boot 后端。

这些目录按照学习顺序编号，方便回顾每个阶段使用的技术和项目结构。
